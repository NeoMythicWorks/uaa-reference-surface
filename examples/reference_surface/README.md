\# Reference Surface



This directory contains the minimal executable proof of UAA enforcement behavior.



\## Files



\- `canonicalize.py` - deterministic canonical action representation  

\- `admissibility.py` - boundary evaluation  

\- `authorization.py` - authorization artifact issuance  

\- `replay\_cache.py` - replay detection  

\- `control\_point.py` - verification immediately before execution  

\- `run\_demo.py` - end-to-end demonstration of allow and block behavior  



\## Run directly



From this directory:





python run\_demo.py





Or from the repository root:





python examples\\demo.py





This is a reference surface, not a production deployment.
